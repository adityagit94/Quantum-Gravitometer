# Scientific notes

The bench side of this project is still a **measurement-like software bench**. It is useful for algorithm comparison, PSD analysis, Allan deviation analysis, and report generation.

The AISim module improves the project by adding a **real atom-optics simulation package** to the workflow. The currently integrated model is a compact **Rabi-oscillation scan**. This is scientifically meaningful as a simulation building block, but it is **not yet** a full gravimeter sequence or an absolute gravity validation.
